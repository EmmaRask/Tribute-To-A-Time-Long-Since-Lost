# Tribute To A Time Long Since Lost

A minimal WordPress theme created as a visual and personal project.

This site explores nostalgia, memory, and visual storytelling through a 
clean and editorial layout. Inspired by the work of lundgrenlindqvist.se, 
specifically their website created for make-up artist Anna De Garcia. 
This project is a personal interpretation and adaptation of that design direction.

## ✧ About the project

The project is built as a custom WordPress theme, focusing on:

* Minimal structure and typography
* Asymmetrical layouts (e.g. the "Seasons" page)
* Image-driven storytelling
* A calm, editorial aesthetic

All images used in this project are original and photographed by me.

## ✧ Structure

The theme is built with a simple and clean setup:

* `front-page.php` – landing page with hero and navigation
* `single.php` – individual project pages
* `page.php` – static pages (e.g. About)
* `style.css` – all styling
* `assets/` – images and media
